RTT

  • sample: http://beej.us/guide/bgnet/html/multi/clientserver.html
  • 1 kilobyte = 1024 char = 1024 byte

server

  • server_setup : parameter is port number
    • get sock file descriptor
    • initialize server address: memset(pointer, 0, sizeof(server_address))
    • initialize address, family, port
    • bind the server address to the socket file descriptor
    • listen to the socket
  • send
    • send the message to the current client
    • ✺ the type of the message if string , -> string.c_str() with the length of the message
  • receive
    • while(1) : the outer while loop is used to accept connections from clients, there is no break in the outer while loop, always prepare to receive messages
      • get the client address by accept(current_socket_file_descriptor, &client_address, sizeof(client_address))
      • convert the client address
      • create a server thread and run a while loop
        • while(1): the inner loop is used to accept messages from the current client
          • if getting the message from client successful then break this inner while loop
            Server running process
  1. set up the server with a port number
  2. create a thread and run the while loop
    1. check if receive a message, if message then send a message to current client
  3. if create the thread successfully
    1. receive the message from the current client in the while loop

Client

  • client_setup: parameters server address and port number
    • check and build the socket
    • check the server address if available
      • if not, get host by name
      • check other available addresses from server’s address list
    • set family and port
    • build connection with server
  • send
    • check socket is available
    • send message and length
  • receive: return the size of message n
    • int recieve(int new_sock_file_descriptor, char* buffer, int max_buffer_size)
    • buffer[n] = 0 ; // set the end signal of message
  • read
    • set a buffer
    • add to string
      Client running process
  1. setup : host address and port number
  2. while loop
    1. send message
    2. receive message
    3. sleep
      char[] buffer
      delete[] buffer
